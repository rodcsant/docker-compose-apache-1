Este eh o meu projeto docker-compose para o bootcamp da DIO Jornada DevOps com AWS DIO - Impulso

Docker compose eh uma ferramenta desenvolvida para ajudar a definir e compartilhar aplicativos com diversos containers. Com o compose, voce pode criar um arquivo YAML para definir os servicos e com um unico comando, pode rodar todos os containers ou para-los.

A versao do Compose precisa acompanhar a versao do Docker, ou seja, eh necessario verificar no site do Docker a versao correspondente.

Os parametros a ser definidos sao: services, environment, ports, volumes e networks.

Para instalar:
apt-get install -y docker-compose

Crie alguns diretorios para nossos arquivos:
mkdir /data
cd /data
mkdir mysql-C
cd /
mkdir /compose
cd /compose
mkdir /first
cd /first

Crie o yaml:
nano docker-compose.yml

Insira no arquivo:
version: '3.8'

services:
  mysqlsrv:
    image: mysql:5.7
    environment:
      MYSQL_ROOT_PASSWORD: "Senha123"
      MYSQL_DATABASE: "testedb"
    ports:
      - "3306:3306"
    volumes:
      - /data/mysql-C:/var/lib/mysql
    networks:
      - minha-rede

  adminer:
    image: adminer
    ports:
      - 8080:8080
    networks:
      - minha-rede

networks: 
  minha-rede:
    driver: bridge
-------------------------------------------
Salve e saia.

Para executar o yaml:
docker-compose up -d
# <filename> up <parameter>
# -d execução em segundo plano

Faça o teste no browser com o IP do host para executar o adminer.
http://[IP]:8080

Servidor: [nome-do-servico-criado]
Usuario: [root]
Senha: [Senha123]
Base de dados: [nome-do-db-criado]

Para encerrar:
docker-compose down
# <filename> down <parameter>
